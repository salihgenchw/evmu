# Evmu

Ev Muhasebesi - Kredi ve Kredi Kartı Borç Takibi

## Moduller

- Sıfır Borç : `Borç Takibi`
- BirAla: `Almak istediğin ve para biriktirdiğin ürünlerin takibi`
- Krediler: `Krediler takibi`
- Portfoy: `Yatırımlarının takibi(Borsa, Kripto)`
